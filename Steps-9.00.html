Create Required Databases in SQL Server 9
*These Steps will be performed in SQL Server Management Studio (SSMS)
  • Right Click on Databases, then click on New Database.
  • Name the Database. 
  • Click Ok to create the Database, then refresh Database to locate New Database Created.


Repeat this step for all three layers we are looking to create.

  • "Incidents_STG"
  • "Incidents_ODS"
  • "Incidents_DW"

Once you create the 3 databases we can move forward to the next step.
